## JD 秒杀小工具

1. 打开工具JD_seckill.exe
2. 登录自己的京东账号－使用扫一扫即可（也不担心账号密码泄漏)
3. 选择想要秒杀的商品进入商品详细页面
4. 选择商品细分参数
5. 选择数量
6. 点击右上角自动秒杀即可
